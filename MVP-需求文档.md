# Image Background Remover - MVP 需求文档

**项目名称**：Image Background Remover  
**版本**：v1.0 MVP  
**创建日期**：2026-03-23  
**负责人**：iris_huang

---

## 1. 项目概述

### 1.1 产品定位
一款极简的在线图片背景移除工具，用户上传图片，即可获得透明背景的高质量图片。

### 1.2 核心价值
- **快**：3秒内完成背景移除
- **简单**：拖拽上传，一键下载
- **免费**：每日有限次数免费使用

### 1.3 目标用户
- 电商卖家（淘宝、拼多多、Shopify）
- 设计师、自媒体创作者
- 需要快速处理产品图的个人卖家

---

## 2. 功能需求

### 2.1 核心功能（MVP必须）

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 图片上传 | 支持拖拽和点击上传 | P0 |
| 背景移除 | 调用 Remove.bg API 处理图片 | P0 |
| 图片下载 | 一键下载透明背景图（PNG） | P0 |
| 预览对比 | 显示原图和处理后对比 | P1 |

### 2.2 用户流程

```
用户访问页面
    ↓
拖拽/选择图片上传
    ↓
自动发送到 Worker API
    ↓
Worker 调用 Remove.bg API
    ↓
返回结果并展示预览
    ↓
用户点击下载
```

### 2.3 不做的功能（本期MVP）
- 用户登录/注册
- 账户余额/充值系统
- 批量处理
- 背景替换
- 历史记录

## 3. 技术方案

### 3.1 技术栈

| 组件 | 技术选型 | 说明 |
|------|----------|------|
| 前端 | Cloudflare Pages + HTML/CSS/JS | 静态托管，无需后端 |
| 后端 | Cloudflare Worker | 处理 API 请求 |
| 图片处理 | Remove.bg API | 第三方背景移除服务 |
| 域名 | 自定义域名（可选） | 后期绑定 |

### 3.2 系统架构

```
┌─────────────┐     ┌──────────────┐     ┌─────────────┐
│   用户浏览器  │────▶│ Cloudflare   │────▶│ Remove.bg   │
│  (上传图片)   │◀────│   Worker     │◀────│    API      │
└─────────────┘     └──────────────┘     └─────────────┘
```

### 3.3 API 设计

#### Worker API Endpoint

**POST /** (处理图片)

Request:
```
Content-Type: multipart/form-data
Body: image (文件)
```

Response:
```
Content-Type: image/png
Body: 处理后的图片二进制数据
```

Error Response:
```
{
  "error": "错误信息",
  "code": "ERROR_CODE"
}
```

### 3.4 环境变量

| 变量名 | 说明 | 示例 |
|--------|------|------|
| REMOVE_BG_API_KEY | Remove.bg API Key | xxx |

## 5. 后端设计

### 5.1 Worker 处理流程

```javascript
1. 接收 multipart/form-data
2. 提取图片文件（限制 10MB）
3. 验证文件类型（jpg/png/webp）
4. 调用 Remove.bg API
5. 返回处理结果
```

### 5.2 错误处理

| 错误码 | 说明 | 用户提示 |
|--------|------|----------|
| FILE_TOO_LARGE | 文件超过10MB | "图片太大了，请上传小于10MB的图片" |
| INVALID_FORMAT | 格式不支持 | "请上传 JPG、PNG 或 WebP 格式的图片" |
| API_ERROR | Remove.bg 接口错误 | "服务繁忙，请稍后再试" |
| RATE_LIMIT | 频率限制 | "操作太频繁，请稍后再试" |
